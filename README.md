# Analytics backend

[![code style: prettier](https://img.shields.io/badge/code_style-prettier-ff69b4.svg)](https://github.com/prettier/prettier)

Simple graphQL mongoDB API.

## Usage

.env:

```
PORT=4000
PRIVATE_KEY=key
NODE_ENV=development
SENDGRID_API_KEY=key
MONGODB_URI=uri
```

```
yarn install
yarn start
```

Database commands:

```
yarn run db:drop
```
